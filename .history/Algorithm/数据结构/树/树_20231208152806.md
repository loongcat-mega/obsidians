#### 递归序，每个节点出现三次
![image.png](https://yaaame-1317851743.cos.ap-beijing.myqcloud.com/undefinedPasted%20image%2020230201163418.png)

对二叉树某结点的x，其先序序列左侧与后续序列右侧交集，是且仅是x的祖先节点

![image.png](https://yaaame-1317851743.cos.ap-beijing.myqcloud.com/undefinedPasted%20image%2020230201214320.png)

前序序列：左兄弟+祖先结点，x，右兄弟+孩子结点
后续序列：左兄弟+孩子结点，x，右兄弟+祖先结点

```c++
void insert(int value)  
{  
    if(root==nullptr)  
        root=new Node(value);  
    else 
   {  
        Node*p=root;  
        bool left_flag=false;  
        bool right_flag=false;  
        while(p&&!left_flag&&!right_flag)  
            if(value>p->data)//往右子树走  
                if(p->right)//右子树还有数据  
                    p=p->right;  
                else           
	                right_flag=true;  
            else if(value<p->data)  
                if(p->left)  
                    p=p->left;  
                else                  
                  left_flag=true;  
            else         
				return;       
		if(right_flag)  
            p->right=new Node(value);  
        if(left_flag)  
            p->left=new Node(value);  
    }  
}
```
